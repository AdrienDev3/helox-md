## HELOX Pair Website (Fix for Not Found)

The previous public URL was not deployed, so it returned `Not Found`.

This repo now includes a built-in pairing website:

- Start locally: `npm run start:webpair`
- Open: `http://localhost:3000`
- API: `http://localhost:3000/pair?phone=25261XXXXXXX`

Deploy this same app to Render using:

- Build Command: `npm install`
- Start Command: `npm run start:webpair`

After deploy, replace:

- `https://YOUR-HELOX-PAIR.onrender.com`

with your real Render URL.
